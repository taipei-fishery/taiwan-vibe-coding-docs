人工訂閱

  1. 來信主旨：訂閱 P1 或 P3 — 公司名稱 — BC Tenant ID。
  2. 附上統一編號（台灣 B2B）與聯絡人。
  3. 我們寄送結帳連結或報價單，付款後寄 API Key 與設定說明。

常見問題

問題回答
一定要買雲端 API 嗎？不必。P2 可不接 Endpoint 使用內建試算；P4 可僅用 Log 管線。
可以自建 Bridge 嗎？可以。依公開 ISV 契約實作 HTTPS 端點，無需訂閱我們的雲端服務。
薪資和發票要一起買嗎？不必。AppSource 上為兩個獨立訂閱。
幣別與發票？AppSource 以 Microsoft 顯示為準（多為 USD）。台灣直售雲端 API 以 TWD 計價，可開立 B2B 電子發票。
