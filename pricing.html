訂閱方案

Microsoft Dynamics 365 Business Central · 台灣在地化 · 產品訂閱

我們採兩層訂閱：在 AppSource 訂閱 BC 延伸模組；需要進階薪資試算或真實電子發票上傳時，可另訂閱雲端 API（或自建 Bridge）。

Layer 1 — Business Central 延伸模組（AppSource）

金流與試用由 Microsoft 處理。各產品獨立訂閱，30 天試用，按公司（租戶）計費。

產品說明月費 (USD)年費 (USD)

Layer 2 — 雲端 API（Taiwan Vibe Coding 直售）

需已安裝對應 BC 延伸模組。於 BC Setup 填入我們提供的 HTTPS Endpoint 與 API Key。

套裝優惠（雲端部分）

  • Payroll Complete：已訂閱 Payroll for BC + Engine API → 雲端月費 USD 39
  • E-Invoice Complete：已訂閱 E-Invoice for BC + Gateway API → 雲端月費 USD 79
  • Taiwan Suite：四項皆備 → 雲端合計 USD 118/月（Engine + Gateway）

如何訂閱雲端 API

  1. 確認已安裝（或試用）對應的 AppSource 延伸模組。
  2. 來信 support@vibe-coding.tw，主旨：訂閱 P1 或 P3 — 公司名稱 — BC Tenant ID。
  3. 我們寄送 API Key、Endpoint URL 與 BC 設定說明。
  4. 完成一筆試算或測試上傳後開始計費（試用期滿後）。

常見問題

問題回答
一定要買雲端 API 嗎？不必。P2 可不接 Endpoint 使用內建試算；P4 可僅用 Log 管線。
可以自建 Bridge 嗎？可以。依公開 ISV 契約實作 HTTPS 端點，無需訂閱我們的雲端服務。
薪資和發票要一起買嗎？不必。AppSource 上為兩個獨立訂閱。
幣別？AppSource 以 Microsoft 顯示為準（多為 USD）；直售雲端 API 合約可另開 TWD 發票。
